# 🚀 Velora AI

Clareza financeira com inteligência artificial.

---

## 📦 Como rodar o projeto

### 1. Baixar o projeto

Clique em "Code" → "Download ZIP"  
ou use:

git clone https://github.com/SEU_USUARIO/SEU_REPO

---

### 2. Abrir o projeto

Abra a pasta no VS Code ou no terminal.

---

### 3. Rodar o backend

No terminal:

cd backend

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

uvicorn app.main:app --reload

---

### 4. Rodar o frontend

Abra outro terminal:

cd frontend

npm install
npm run dev

---

### 5. Acessar o sistema

Abra no navegador:

http://localhost:5173

---

## 🔒 Privacidade

Seus dados ficam apenas na sua máquina.  
Nada é enviado para servidores externos.
